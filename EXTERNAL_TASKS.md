# External Launch Tasks

These items require access to external accounts and cannot be completed inside the repository alone.

## Search Indexing

1. Add `https://foshchii.github.io/` to Google Search Console.
2. Submit `https://foshchii.github.io/sitemap.xml`.
3. Optional: submit or share the article feed `https://foshchii.github.io/feed.xml` anywhere that accepts RSS feeds.
4. Request indexing for:
   - `https://foshchii.github.io/`
   - `https://foshchii.github.io/resume.html`
   - `https://foshchii.github.io/projects.html`
   - `https://foshchii.github.io/insights.html`
   - the three insight articles under `/insights/`
5. Add the site to Bing Webmaster Tools and submit the same sitemap.

## Analytics

Pick one privacy-friendly provider before adding a production tracking script:

- Cloudflare Web Analytics
- Plausible
- GoatCounter

Recommended: Cloudflare Web Analytics if the domain is already managed there; otherwise Plausible or GoatCounter.

## Promotion

1. Add the website to the LinkedIn profile featured section.
2. Share each insight article as a short LinkedIn post.
3. Add the site and best articles to the GitHub profile README.
4. Link the resume page directly when applying for roles.
